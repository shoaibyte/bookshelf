# Digital Bookshelf

A categorized collection of PDF books for learning and reference. All books live under `bookshelf/` and are organized by topic.

## Structure

- **Root**: `bookshelf/` â€” every book or multi-file title is placed here.
- **Categories**: Each category is a subfolder (e.g. `bookshelf/Software Engineering/`).
- **Multi-file books**: Some titles are split into chapters or parts and live in their own subfolder inside the category (e.g. `bookshelf/Software Engineering/Chaos_Engineering/`).

## Category index

| Category | Description |
|----------|-------------|
| **Software Engineering** | Programming (Go, Java, Scala), concurrency, databases, distributed systems, messaging (Kafka, RabbitMQ), compilers/interpreters, chaos engineering, design (DDD, LLD), and related tech books. |
| **History** | World history, great maps and cartography. |
| **Geography and Maps** | Atlases and geography reference. |
| **Productivity and Self-Improvement** | Focus, productivity, and personal effectiveness. |
| **Mathematics and Science** | Math and science learning. |
| **Reference and General Knowledge** | General reference and how-things-work. |
| **Religion and Culture** | Religion and culture. |
| **Aviation** | Aviation history. |
